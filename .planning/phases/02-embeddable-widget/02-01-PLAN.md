Create `packages/widget/` directory and all configuration files:

1. **package.json**: name `roofing-widget`, private, scripts: `dev` (vite), `build` (vite build), `test` (vitest run). Dependencies: `preact@^10`, `@preact/signals@^1`. DevDependencies: `vite@^6`, `@preact/preset-vite@^2`, `typescript@^5.7`, `vitest@~2.1`, `jsdom` (for Shadow DOM testing).

2. **tsconfig.json**: target ES2020, module ESNext, moduleResolution bundler, jsx react-jsx, jsxImportSource preact, strict true, include `["src"]`.

3. **vite.config.ts**: Use `@preact/preset-vite` plugin. Build in library mode: entry `src/index.ts`, name `RoofingWidget`, fileName `() => 'roofing-widget.js'`, formats `['iife']`. Set `cssCodeSplit: false`, `rollupOptions.output.inlineDynamicImports: true`. Minify with terser, keep console.error.

4. **vitest.config.ts**: Extend vite config, set environment to `jsdom`, include `['test/**/*.test.ts']`.

5. **index.html**: Dev-only test harness. Simple HTML page that loads the widget script with `data-company-id="test-company-1"`. Include a note that this is for development only.

6. **src/state/form.ts**: Create Preact signals state module:
   - `currentStep` signal (0=roof, 1=contact, 2=estimate)
   - `formData` signal with fields: sqft (string), pitch (string), material (string), firstName (string), lastName (string), email (string), phone (string), consent (boolean=false)
   - `estimateResult` signal (null or {estimateLow, estimateHigh, disclaimer})
   - `isLoading` signal (boolean)
   - `updateField(field, value)` helper that spreads into formData
   - `nextStep()` / `prevStep()` helpers that clamp step range [0, 2]

7. **src/api/client.ts**: Create API client module:
   - `apiBase` variable: derive from `document.currentScript` src attribute by stripping the `/widget/roofing-widget.js` path, falling back to empty string (same origin) for dev.
   - `fetchCompanyConfig(companyId: string)` -> fetches `GET {apiBase}/api/config/{companyId}`, returns parsed JSON.
   - `submitEstimate(data)` -> fetches `POST {apiBase}/api/estimates` with JSON body, returns parsed JSON. Accept all fields (sqft, pitch, material, companyId, plus optional firstName, lastName, email, phone, consent).

Run `cd packages/widget && npm install` after creating package.json.
