   ```
   adminSessions = sqliteTable('admin_sessions', {
     id: text('id').primaryKey(),        // nanoid session token
     companyId: text('company_id').notNull().references(() => companies.id),
     expiresAt: text('expires_at').notNull(),
     createdAt: text('created_at').default("(datetime('now'))"),
   })
   ```

2. **Bindings (packages/api/src/types.ts):** Add `LOGOS_BUCKET: R2Bucket` and `API_BASE_URL?: string` to the Bindings type. The R2 bucket is used in Plan 02 but adding now avoids touching this file twice.

3. **Password helpers (packages/api/src/auth/password.ts):** Create `hashPassword(password: string): Promise<string>` and `verifyPassword(password: string, stored: string): Promise<boolean>` using Web Crypto API PBKDF2 (100,000 iterations, SHA-256, 16-byte random salt). Store as `base64salt:base64hash`. Do NOT use bcrypt -- it does not work in Cloudflare Workers.

4. **Session helpers (packages/api/src/auth/session.ts):** Create three functions:
   - `createSession(db, companyId): Promise<string>` -- generates nanoid(32) token, inserts into adminSessions with 7-day expiry, returns token
   - `validateSession(db, token): Promise<string | null>` -- looks up token in adminSessions, returns companyId if valid and not expired, null otherwise. Delete expired sessions on lookup.
   - `deleteSession(db, token): Promise<void>` -- deletes session row

5. **Auth middleware (packages/api/src/middleware/auth.ts):** Create Hono middleware using `createMiddleware<{ Bindings: Bindings }>`. Read 'session' cookie via `getCookie(c, 'session')`. Call `validateSession`. If invalid, return 401 JSON `{ error: 'Unauthorized' }`. If valid, set `companyId` on context via `c.set('companyId', companyId)`. Use Hono Variables typing: define `type AdminVars = { companyId: string }` and export it.
